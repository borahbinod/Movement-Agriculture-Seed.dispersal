# Movement, Agriculture and Seed Dispersal

## Project Overview

This project investigates how agricultural landscapes influence animal movement and the resulting seed dispersal patterns. Using simulated landscapes and individual-based movement models, the project evaluates how varying levels of riparian forest cover and fragmentation affect individual movement trajectories and behavior, and seed deposition across landscapes.

### Objectives

1. Simulate landscapes with varying forest cover and fragmentation.
2. Simulate the movement of seed-dispersing birds across landscapes, accounting for their
   habitat preferences as well as propensity to travel or forage/rest in different habitats.
3. Simulate seed depositions resulting from movement and quantify resulting seed dispersal patterns.
4. Evaluate how forest loss and fragmentation alter dispersal processes.

## Afrotropical_bird.csv

| Variable | Description |

| Species | Bird species |
| Body_mass | Body mass |
| Movement_distance | Average movement distance |
| Home_range | Home range size |
| Diet | Dietary classification |
| Dispersal_capacity | Seed dispersal ability |

### Source

Species trait data compiled from published literature and databases.

### Usage

Used as input for movement simulations and dispersal parameterization.

# Code Documentation

## landscape_sims.R

### Purpose

Generate simulated landscapes representing different levels of forest loss and fragmentation.

### Inputs

- Afrotropical_bird.csv

### Outputs

Files written to:

```text
Data_derived/
```

### Main Steps

1. Create landscape grid.
2. Assign habitat classes.
3. Vary proportion of forest cover and degree of fragmentation.
4. Calculate landscape characteristics.
5. Save simulated landscapes.

### Ecological Interpretation

Forest loss and fragmentation alter habitat availability and connectivity, potentially affecting animal movement, behavior, and seed dispersal.

---

## Individual_movement_simulations.R

### Purpose

Simulate movement trajectories of individual birds across landscapes generated by `landscape_sims.R`.

### Inputs

- Simulated landscapes
- Species traits from Afrotropical_bird.csv

### Outputs

Files written to:

```text
Output/
```

### Main Steps

1. Initialize individual location.
2. Determine movement direction.
3. Determine movement distance.
4. Update location.
5. Record trajectory.
6. Simulate seed deposition events.

### Ecological Interpretation

Movement trajectories determine where seeds are dispersed across heterogeneous landscapes.

# Assumptions

The simulations assume:

1. Movement behavior is adequately represented by specified movement rules.
2. Simulated landscapes represent realistic agricultural gradients.
3. Species traits accurately reflect dispersal behavior.
4. Seed deposition probability is linked to movement behavior.

---

# Limitations

1. Simulated landscapes simplify real-world habitat complexity.
2. Behavioral responses may vary among individuals.
3. Environmental stochasticity may not be fully represented.
4. Species interactions are not explicitly modeled.
